Financisto charts

Set the data

If you are a Financisto user, on that page you can generate some charts to see how your costs were changed in recent times. You will need to create a backup file in your Financisto app, and upload somewhere where it's publicly available to enable to download it. For example you can use your Dropbox account: copy a backup file into your public folder and use it's url to generate charts.
If you can't or don't want to put into a public place, you can also upload it's content by yourself. First switch to the "Copy from uncompressed backup file" tab below, then unzip the backup file and open it in a text editor then copy its content into the text area on this page.
